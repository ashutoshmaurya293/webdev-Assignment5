Q.7 What’s async & await Keyword in JavaScript
Answer: async and await is a way to write asynchronous code The async keyword is used to define an asynchronous function and An asynchronous function is a function that returns a Promise and The await keyword is used to wait for a Promise to resolve before continuing execution of the code.
